# Your Name - Portfolio

A professional portfolio site to showcase your work and experience.

## Features

- Clean, dark-themed design
- Responsive layout
- Sections: About, Skills, Projects, Experience, Contact
- Single HTML file - easy to deploy anywhere

## Deployment

Deploy to GitHub Pages or any static hosting:

### GitHub Pages
1. Push this repo to GitHub
2. Go to Settings > Pages
3. Select your branch and save

### Other Hosting (Netlify, Vercel, etc.)
Simply deploy the `index.html` file - no build step required.

## Customization

Edit `index.html` to add your own information.

## Tech Stack

- HTML5 & CSS3
- Google Fonts (Inter, JetBrains Mono)
- Inline SVG icons (no external dependencies)
